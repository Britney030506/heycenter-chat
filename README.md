# HEYCENTER CHAT UI

This is a UI Vue project of a support chat app, full responsive with dark mode and more features!

## Installation

Use npm to install the project dependencies.

```bash
npm install
```
## Usage
To start the development server, run the following command:
```bash
npm run dev

```
